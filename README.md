# Mortgate Calculator Web App

## Pre-requisites

* Node v12+
* Npm v6+
* React v16+
* Postman

## Get Started

* To run locally

`npm install`
`npm run start`

## Deploying to Firebase

* Make sure you build with the latest changes

`npm run build`
`firebase deploy`

* Navigate to the hosting URL shown in the cmd

`https://sheltered-escarpment-94741.web.app/`


## Testing

`npm run test`
